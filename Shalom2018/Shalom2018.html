Defining Intraindividual Variability in Symptoms

Sum Scores versus Items

# Load packages ----
library(tidyverse)
library(patchwork)

Shalom et al. (2018) definition of intraindividual variability in symptoms

Shalom et al. (2018) define intraindividual variability in symptoms as weekly changes in sum score of multiple item measures, see formula below. When predicting sudden gains by intraindividual variability prior to the gain, the authors mention that this measure was only calculated when four or more session by session intervals were available to increase the reliability. As a results sudden gains that occurred early in treatment (before session 5) were excluded from this particular analysis. In studies where a substantial subgroup of participants experience gains early in treatment, this may be problematic. This is of course less problematic when pretreatment intraindividual variability is used as a predictor for sudden gains (Shalom et al. 2020).

\[\sum^{s=N}_{s=1}\left(\frac{|\Delta_{s},s+1|}{N}\right)\]

Create example data (item by item)

Building on results from their earlier study Shalom et al. (2020) investigate pretreatment and within-treatment intraindividual variability in the sum scores of two symptom measures that were administered weekly: (a) the severity of social anxiety was measured using the 17-item Social Phobia Inventory (SPIN) and (b) depressed mood was measured using the 9-item version of the Patient Health Questionnaire (PHQ-9).

Shalom et al. (2018) definition of intraindividual variability focuses on the severity of the main outcome measure but does not account for item by item symptom changes. Depending on the research question and definition of intraindividual variability in symptoms both methods may be valid, but potentially measure different aspects. I demonstrate this problem in the following example:

First, I create a dataset with 5 participants who were assessed with a 2-item measure during 7 sessions.

  • id: ID variable
  • n: Indicates an event, in this case “sudden gain”
  • x_s1_i1 - x_s7_i2: Item by item measures

Calculating sum scores

Next, I’m calculating sum scores for each participant at every session. Note that this a very basic example data set with no missing values. If there were missing values, this code would have to be changed depending on whether sum scores should be prorated or not.

# Create list of variable names
var_names_item <- c("x_s1_i1", "x_s1_i2", 
                    "x_s2_i1", "x_s2_i2", 
                    "x_s3_i1", "x_s3_i2", 
                    "x_s4_i1", "x_s4_i2", 
                    "x_s5_i1", "x_s5_i2", 
                    "x_s6_i1", "x_s6_i2", 
                    "x_s7_i1", "x_s7_i2")

data_sum <- data %>% 
  pivot_longer(cols = all_of(var_names_item), 
               names_to = c("measure", "session", "item"),
               names_pattern = "(.)_(.*)_(.*)") %>% 
  group_by(id, measure, session) %>% 
  mutate(sum = sum(value)) %>% 
  select(id, n, measure, session, sum) %>% 
  distinct() %>% 
  pivot_wider(names_from = c(measure, session), values_from = sum)

This code is counting the “variability” in sum scores prior to session ‘n’ using slice(c(1:unique(n))).

data_change_sum <- data_sum %>% 
  filter(n > 4) %>%
  pivot_longer(all_of(var_names_sum),
               names_to = c("measure", "session"),
               names_pattern = "(.)_(.*)") %>% 
  mutate(session = as.numeric(str_extract(session, "\\d+"))) %>% 
  group_by(id, measure) %>%
  slice(c(1:unique(n))) %>% 
  mutate(diff_abs = abs(value - lag(value))) %>% 
  mutate(lag_value = lag(value),
         diff_value = value - lag_value,
         value_change = sum(diff(sign(diff_value)) != 0, na.rm = T)) %>% 
  ungroup() %>% 
  select(id, n, measure, value_change) %>% 
  distinct()

Conclusion

This theoretical example demonstrates that Shalom et al. (2018) definition of intraindividual variability might miss aspects of symptom changes on an item by item level. Depending on the definition and research question this could actually be desired. Only of variability on an item by item level is of theoretical interest it would be important to consider an adaptation of the method proposed by Shalom et al. (2018).

For example id = 5 shows no variability in the sum scores but most variability in item scores compared to other participants. One potential way to address this would be to include item by item changes in the formula instead of changes in sum scores as indicated by the index i in \(\Delta_{s_i}\), see the adapted formula below:

\[\sum^{s=N}_{s=1}\left(\frac{|\Delta_{s_i},s+1|}{N}\right)\]
